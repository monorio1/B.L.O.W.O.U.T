# B.L.O.W.O.U.T

**BURN’s Level Optimization & World Overhaul Utility Tool**

---

## Overview

**B.L.O.W.O.U.T** is a map-fixing and optimization mod for *S.T.A.L.K.E.R. Anomaly*.
It consolidates and expands upon existing community fixes while introducing additional improvements to rendering and performance.

This project integrates work from:

* Expanded/Fixed Maps: All In One (by HailTheMonolith)
* C.B.R Overhaul (by VodkaPivin)

---

## Features

### Foliage Lighting Fixes

* Corrects lighting on trees and large foliage assets
* Addresses inconsistencies across multiple maps
* Improves visual coherence without altering gameplay

### Occlusion & Performance Improvements

* Adds and refines occluders on problematic maps
* Improves culling behavior
* Reduces unnecessary rendering load
* Results in measurable performance gains depending on scene complexity
